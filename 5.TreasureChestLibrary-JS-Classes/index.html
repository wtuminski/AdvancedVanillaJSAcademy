Treasure Chest

All of the magic here happens in the console.

You may access the library by creating a new instane by writing `new TreasureChest()` in the console and using one of follwing methods or properties: bronze, silver, gold, addBronze, addSilver, addGold or getLoot. All of the adding methods might be chained. The constructor TreasureChest has also a static method getRandomLoot.

You may also provide options with base amount of gold, silver or bronze and custom function `getCustomGetLootMessage` that returns customized string in getLoot method.

Example:

new TreasureChest({ gold: 20, bronze: 5, getCustomGetLootMessage: function(){ return `This the amount of your shitty gold: ${this.gold}`; } })
